# ZJSUMessage
## 项目计划

移动Web开发-基于组织架构的聊天系统——商大联系通

### 计划实现功能：

1.能够完成基础的即时通讯

2.能够填写在线文档

3.能够完成表单和文件提交

4.能够发布基于web的直达功能
